# Resin by ATS

Resin is an enterprise-grade workflow automation platform built specifically for Africa. Forked and heavily localized by ATS (Akinwunmi Technology Services), Resin provides seamless integrations with local infrastructure including Paystack, Flutterwave, and CBN systems.
